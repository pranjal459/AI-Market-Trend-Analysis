

**Retail Sales Forecasting & Customer Behavior Prediction**

## Project Overview

This project uses machine learning to analyze retail transaction data and generate business insights through:
- Customer segmentation (K-Means)
- Sales forecasting (Prophet)
- Churn prediction (Random Forest)

## Dataset

- Source: UCI Online Retail Dataset
- Size: 400,000+ transactions
- Period: 2010-2011
- Link: [UCI Repository](https://archive.ics.uci.edu/ml/datasets/Online+Retail)

## Technologies Used

- Python 3.10
- pandas, NumPy
- scikit-learn
- Prophet
- Matplotlib, Seaborn, Plotly
- Jupyter Notebook
